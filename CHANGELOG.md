# Changelog

All notable changes to OpenAPI Validator TUI will be documented in this file.

## v0.1.0

- Add custom generator support via YAML definitions
- Use docker-compose for compile phase
- Add configurable keybindings via `.oavc` config
- Add `.oav/` scaffolding, config validation, log persistence
- Add built-in generator registry and config passthrough
- Add generation diff view to compare output across pipeline runs
- Add generated code browser view
- Add syntax highlighting for spec context panel
- Add in-TUI fix proposal workflow
- Add external editor integration and cancel-on-re-run
- Add complete keybindings and graceful degradation
- Add validation pipeline orchestrator
- Add panel layout with functional List widgets and status bar
- Add spec parser with JSON pointer to line number mapping
- Add lint log parser for Spectral/Redocly stylish output
- Add docker orchestration module with streaming and cancellation
- Add config loading and spec discovery
- Scaffold project with ratatui TUI skeleton
